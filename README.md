# GamingBall
A 3d visualization of a football and a basketball

You can change the texture of the ball whenever you want (there is a deidcated button to do it)

## Features
- The website offer a scroll oriented camera
- contains different texture for playing balls
- A dedicated summary of the basketBall game
- Astonishing 3d models and a basketball court background
